JFinalUIB现在分为两个版本:

JFinalUIB_V1版本：基于JFinal1.9 + charisma UI（老版本，几乎停止维护）
JFinalUIB_V2版本：基于JFinal2.2 + rapido UI

GIT版本分支一共有3个：

	master：JFinalUIB_V2代码，基于常规的eclipse和idea项目配置文件构建
	JFinalUIB_V2_MAVEN：基于maven环境构建的JFinalUIB_V2，代码进行了maven模块化切分
	JFinalUIB_V1：老版本，几乎停止维护

V2主要变动：

	升级JFinal为2.2、重构项目
	更换UI,新的UI库是rapido UI
	更多数据库支持，现支持PostgreSQL、MySQL、SqlServer、Oracle、DB2
